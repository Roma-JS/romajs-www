romajs-www
==========

The RomaJS homepage.

Including:
*    [Assemble.io](http://assemble.io)
*    [Zurb Foundation](http://foundation.zurb.com)

Getting started
---------------------
### Installing dependencies
```
npm install
bower install
```

## Launch dev server
```
grunt server
```

## Compile dist code
```
grunt build
```
